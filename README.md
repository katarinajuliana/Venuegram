Venuegram
====  

A web app that shows Instagram posts for given Foursquare venues selected by
the user on a Google Map.

**Technologies used:**  
• Ruby 2   
• Rails 4  
• CoffeeScript  
• Turbolinks  
• Slim  
• Instagram API  
• Google Maps API  
• Foursquare API  
• RestClient  
• RSpec  
• Capybara  
• Twitter Bootstap 3  
• jQuery  


"bundle exec rspec" to run test suite
